# Email Notification Feature

Fitur ini digunakan untuk:

- Mengirim email notifikasi booking
- Mengirim email approval booking
- Mengirim email pengembalian equipment

Workflow:

1. User melakukan booking
2. Sistem mengirim email notifikasi
3. Admin menerima informasi booking
4. User menerima status booking realtime

Status:
Development in progress
